# IMG421Proj3
### Description
This project implements boids that are customizable in their following/grouping behavior, how many spawn, colors, etc. This is simply the base game implementation where boids follow an attractor with no further implemented behaviors. The game includes a start menu with a description of the game and a start button to begin viewing the boids.
